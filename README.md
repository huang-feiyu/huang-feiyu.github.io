# Huang Blog

> 本博客为中英文双语博客（以中文为主），不对两者进行区分。

## 目的

* **为个体写作**
* 传播知识
* 隐含地展现自我
* 向读者传达真正有价值的想法

## 分类

* Series:  系列文章
* Essay:   哲学散论
* Review:  评论性文章
* Journal: 个人日志 (不长期维护)
* Other:   其他文章
